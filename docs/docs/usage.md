# Uso

1. Instale as dependências com `pip install -r requirements.txt`.
2. Execute a aplicação Flask com `python flask_app.py`.
3. Abra o navegador em `http://127.0.0.1:5000/`.
4. Cadastre tarefas informando um título e, opcionalmente, uma data.
5. Remova tarefas com o botão correspondente na lista.
