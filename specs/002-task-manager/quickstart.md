# Quickstart: Task Manager

## Prerequisites

- Python 3.11+
- pip
- A modern web browser

## Setup

1. Create and activate a virtual environment.
2. Install the dependencies for the web app and MkDocs.
3. Run the development server.

## Validation Scenarios

### Create a task

1. Open the application.
2. Submit a task title and optional reminder.
3. Confirm that the task appears in the list.

### Remove a task

1. Click the remove action for an existing task.
2. Confirm that it disappears from the list.

### Reminder behavior

1. Create a task with a reminder time close to the current time.
2. Wait for the reminder time to arrive.
3. Confirm that a visual alert or notification is displayed.
