# Quickstart: To-Do List

## Prerequisites

- A modern web browser
- The project files available locally in the repository root

## Setup

1. Open the repository root in a browser using the file system, or serve the folder with a simple local server.
2. Open the page that loads the app entry point.

## Validation Scenarios

### Add a task

1. Enter a task description in the input field.
2. Submit the form.
3. Confirm that the new task appears in the list.

### Mark a task as completed

1. Click the action for an existing task.
2. Confirm that the task is visually struck through and marked as completed.

### Delete a task

1. Click the delete action for a task.
2. Confirm that the task disappears from the list.

### Persist after reload

1. Add one or more tasks.
2. Reload the page.
3. Confirm that the tasks are still present in the list.

### Invalid input handling

1. Try submitting an empty task name.
2. Confirm that the task is not added.
3. Try adding the same task twice.
4. Confirm that the second attempt is rejected.
